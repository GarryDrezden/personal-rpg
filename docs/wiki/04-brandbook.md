# Brandbook

## Core identity

**Личная RPG** — взрослая personal RPG о возвращении тела, движения, контроля, ясности и устойчивости.

Не приложение для подсчёта калорий. Не медицинский трекер. Не детская игра.

## Tone

- поддерживающий;
- взрослый;
- честный;
- не медицинский;
- не обвиняющий;
- немного dark fantasy;
- без детской мультяшности.

**Запрещённые формулировки в UI:** провал, сорвался, наказание, ты плохой, shame language.

## Visual direction

- stylized realistic 3D game art;
- realistic anatomy base;
- 30–40% stylization;
- premium mobile RPG;
- transparent background for character assets;
- dark fantasy without horror;
- cozy/light secondary theme.

## Themes

| Тема | Роль |
|------|------|
| **Dark Fantasy** | Основная RPG-линейка, journey, codex |
| **Cozy / Light** | Вторичная, настройки «Светлая уютная» |

Яркая male-генерация (розовая рубашка, жёлтые шорты) — **cozy candidate**, не dark fantasy.
