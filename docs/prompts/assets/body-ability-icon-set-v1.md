# Asset: Body Ability Icon Set v1 (12 icons)

## Purpose

Единый набор из **12 согласованных иконок** для Body Abilities v1 (`/growth/abilities`). RPG tokens / badges — наблюдения о теле в быту, не медицинские пиктограммы.

## Style

- stylized realistic 3D game art
- dark fantasy + cozy
- premium mobile RPG feel
- cinematic lighting on each icon
- readable at **48–64px** mobile
- not horror
- not gore
- not anime
- not flat vector
- not corporate illustration
- **not hospital / medical cross / stethoscope style**

## Composition

- **Format:** 12 separate **1:1** icons, shared visual system
- **Frame:** subtle stone or enamel medallion rim, gold-ember accent
- **Lighting:** consistent top-left warm key light across set
- **Style unity:** same material language (aged metal + soft glow)

## Mood

Спокойная сила. «Я заметил в жизни» — не диагноз, не героизм. Каждая иконка — маленькая победа наблюдения.

## Details

- Categories color hint (subtle): mobility=teal ember, endurance=green, daily=amber, confidence=violet, clothing=rose, recovery=blue — all muted
- No human faces required — symbols and objects preferred
- Slight wear on metal — lived-in RPG inventory feel

## Format

- **Target:** `.webp` per icon
- **Aspect:** 1:1 (512×512 master each)
- **Background:** transparent PNG/WebP preferred for UI overlay

## Negative prompt

medical cross, hospital, syringe, anatomy diagram, muscle diagram, anime eyes, chibi, flat vector, corporate pictogram, horror, gore, text, watermark
