# Asset: Пожиратель Ресурса (Season 5 mini-boss)

## Purpose

Сезонный мини-босс 5 — образ **истощения ресурса**: силы уходят быстрее, чем возвращаются.

## Distinct visual identity

**Персонаж-босс** (не здание): рыцарь/оболочка с вакуумной пастью в груди, которая **втягивает** синие потоки энергии из пустых колб.  
Не капюшон+рука (S01–S03), не туманный столб (S04), не алтарь/колодец-архитектура.
