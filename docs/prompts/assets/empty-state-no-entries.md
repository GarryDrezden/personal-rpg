# Asset: Empty state — нет записей

## Purpose

Пустые списки и экраны без данных (Dashboard, Measurements, история дней). Сообщение: «здесь появятся первые следы» — без стыда и грусти.

## Style

- stylized realistic 3D game art
- dark fantasy + cozy
- premium mobile RPG feel
- cinematic lighting
- readable at mobile size
- not horror
- not gore
- not anime
- not flat vector
- not corporate illustration

## Composition

- **Format:** 4:3 or 1:1 card illustration
- **Scene:** пустая тропа в сумерках, маленький **тлеющий костёр** слева или по центру
- **Props:** открытый **дневник** на камне, пустые страницы с мягким свечом
- **Path:** следы ещё слабые или только начинают проявляться — не «пустота провала», а «место ждёт»
- **Sky:** мягкий twilight, одна тёплая звезда или луна

## Mood

Приглашающий, спокойный. «Первый шаг ещё впереди — и это нормально». Поддержка, не оценка.

## Details

- Тёплый свет костра и дневника
- Никаких перечёркнутых календарей, грустных лиц, слёз
- Лёгкий туман для глубины, не депрессивный мрак
- RPG-атмосфера: камень, мох, деревянная обложка дневника

## Format

- **Target:** `.webp`
- **Aspect:** 4:3 (1200×900) — scales in empty-state cards
- **Background:** soft dark gradient compatible with UI cards

## Negative prompt

sad face, crying, shame, failure, red X, warning sign, horror, gore, anime, flat icon, corporate empty state, clipboard medical, hospital, text, watermark

`public/game-assets/ui/empty-no-entries.webp`
