# Asset: Empty state — нет записей

## Purpose

Пустое состояние, когда ещё нет дневных записей — `Dashboard`, `Measurements`, история дней, generic lists. Сообщение: «здесь появятся первые записи». Не стыд, не провал, не пустота наказания — начало пути.

## Style

- stylized realistic 3D game art
- dark fantasy + cozy
- premium mobile RPG feel
- cinematic lighting
- readable at mobile size
- not horror
- not gore
- not anime
- not flat vector
- not corporate illustration
- no text
- no UI

## Composition

- **Format:** 16:9 landscape (preferred for card/banner usage); 4:3 acceptable fallback
- **Scene:** небольшой тёмный уютный уголок — каменная ниша или край тропы
- **Props:** закрытый или пустой **дневник** на камне; слабый **огонёк** (ember) рядом
- **Ground:** лёгкие следы на пыли или камне — едва намеченные, «первый шаг ещё впереди»
- **Light:** мягкий тёплый свет от огня и обложки дневника на холодном graphite фоне

## Mood

Приглашающий, спокойный. «Дневник ещё чистый — костёр ждёт первого следа». Поддержка, не оценка. Здесь появятся первые записи.

## Details

- Тёплый ember + холодные тени угла
- Никаких перечёркнутых календарей, грустных лиц, слёз, красных предупреждений
- Лёгкий туман для глубины — не депрессивный мрак
- RPG-атмосфера: камень, мох, деревянная обложка дневника

## Format

- **Target:** `.webp`
- **Aspect:** 16:9 (1600×900 master recommended)
- **Background:** soft dark gradient compatible with UI cards

## Negative prompt

sad face, crying, shame, failure, red X, warning sign, horror, gore, blood, anime, flat icon, corporate empty state, clipboard medical, hospital, text, watermark, logo, punishment, void abyss
