# Asset: Туманный Барон (Season 4 mini-boss)

## Purpose

Сезонный мини-босс 4 — образ **тумана усталости**, скрывающего ясность. Сопротивление маршруту, не стыд.

## Distinct visual identity

**Не** копия «капюшон + рука + ember по центру» (как S01–S03).  
Босс = **столб / стена живого тумана** с короной вдали; дорога исчезает в тумане; два фонаря у стол.

## Style

- stylized realistic 3D game art, dark fantasy + cozy, premium mobile RPG
- clean smooth digital painting (no film grain / ripple / particle swarm)
- cold ash-blue / silver fog palette
- readable at small boss card size

- **Motif:** two iron lanterns on ruined posts; path swallowed by fog; vague watchtower
- **Hope crack:** tiny warm glow in left lantern only
- **Environment:** mountain pass at twilight, soft vignette
