# Личная RPG

**Локальный трекер здоровья и привычек в формате персональной RPG.**  
Отмечайте квесты дня, копите XP и монеты, прокачивайте навыки, побеждайте недельных боссов и получайте достижения — всё хранится на вашем компьютере.

> **Версия:** v1.4 · **Репозиторий:** [github.com/GarryDrezden/personal-rpg](https://github.com/GarryDrezden/personal-rpg)

## Что это за проект

**Личная RPG** превращает заботу о себе в игру:

- каждый день — набор **квестов** (калории, шаги, трезвость, привычки);
- за выполнение начисляются **XP** (опыт) и **монеты**;
- XP повышает **уровень** персонажа;
- привычки прокачивают **навыки** (Тело, Контроль, Ясность и др.);
- неделя завершается боем с **недельным боссом**;
- долгосрочный прогресс виден на **карте**, в **отчётах** и **достижениях**;
- монеты тратятся в **магазине наград** на то, что мотивирует лично вас.

Приложение **полностью локальное**: данные лежат в SQLite на вашем ПК, интернет не нужен.

## Ключевые понятия

| Понятие | Описание |
|--------|----------|
| **Квест** | Задача на день: основная (калории, шаги, алкоголь) или второстепенная (зарядка, зал, хобби и т.д.) |
| **XP (баллы)** | Опыт за выполненные квесты и недельные бонусы. Суммируется в общий уровень |
| **Монеты 🪙** | Валюта за хорошие дни и недели. Тратится на награды из магазина |
| **Уровень** | Растёт по накопленному XP (пороги: 0 → 500 → 1200 → 2200 → 3500…) |
| **Навык** | Отдельная ветка прогресса: Тело, Контроль, Ясность, Быт, Зелень, Радость |
| **Босс недели** | Случайный недельный вызов с условиями и наградой за победу |
| **Достижение** | Разблокируется за вехи: сброс веса, серии дней, комбо-привычки и др. |
| **Карта** | Визуальный путь по весу, шагам, трезвости, залу и замерам |
| **Аватар** | Персонаж из 7 стадий, меняется по сброшенному весу |

## Быстрый старт для нового пользователя

1. **Запустите приложение** (см. [Установка и запуск](#установка-и-запуск)).
2. Откройте **Настройки** → задайте целевой вес, калории, шаги и цель XP на неделю.
3. При желании настройте **второстепенные цели** (можно заменить «Хобби» на «Йогу» или добавить свои).
4. Каждый вечер заходите в **Сегодня** → отметьте квесты → нажмите **«Сохранить день»**.
5. Раз в неделю (лучше в понедельник) делайте **замеры** и смотрите **Отчёты**.
6. Тратьте накопленные **монеты** в **Награды** на то, что вас мотивирует.

> **Важно:** изменения на странице «Сегодня» не сохраняются сами — нужно нажать **«Сохранить день»**.  
> В **Настройках** изменения применяются кнопкой **«Сохранить»** вверху страницы (кроме темы — она сохраняется сразу).

## Ежедневный сценарий

```
Утро/день          →  Сегодня: внести калории, шаги, отметить привычки
Вечер              →  Сохранить день → получить XP и монеты
Понедельник        →  Замеры + посмотреть отчёт за прошлую неделю
По мере накопления →  Награды: купить желаемое за монеты
```

**Статусы дня** (по сумме XP за день):

| XP за день | Статус |
|-----------|--------|
| ≥ 100 | Отличный день |
| ≥ 70 | Хороший день |
| ≥ 40 | Нормально |
| < 40 | День выживания |

## Разделы приложения

Навигация: **боковое меню** (десктоп) или **нижняя панель** (мобильные). Полный список разделов — в сайдбаре.

### Главная `/`

Центральная панель прогресса. Здесь собрано всё важное:

| Блок | Что показывает |
|------|----------------|
| **Персонаж** | Аватар, уровень, XP, монеты, сброшенный вес |
| **Прогресс уровня** | Сколько XP до следующего уровня |
| **Статус сегодня** | Оценка текущего дня |
| **Квесты дня** | Краткий список с переходом на «Сегодня» |
| **Неделя** | XP за неделю и % от цели |
| **Босс недели** | Текущий вызов и прогресс |
| **Банк наград** | Баланс монет |
| **Навыки** | Краткий обзор прокачки |
| **Карта / Отчёт / Аналитика** | Превью с ссылками |
| **Восстановление** | Появляется после плохого дня — мягкий режим |
| **Достижения** | Последние разблокированные |

### Сегодня `/today`

**Главный экран ввода данных за день.**

**Основные квесты** (обязательный фокус):

| Квест | Как выполнить |
|-------|---------------|
| 🍽️ Калории в лимите | Ввести калории ≤ дневного лимита |
| 👟 Шаги | Ввести шаги ≥ дневной цели |
| 💧 День без алкоголя | Выбрать «Без алкоголя» |

**Средние квесты** (по умолчанию): зарядка, дневник, зал.

**Бонусные квесты** (по умолчанию): готовка, ремонт, цветы, хобби.

- Нажмите на карточку квеста, чтобы отметить выполнение или ввести значение.
- Внизу — **превью XP и монет** за день.
- Кнопка **«Сохранить день»** записывает всё в базу.
- **«Сбросить»** — удалить запись за сегодня (с подтверждением).

После **плохого дня** основные квесты заменяются на **минимальный набор** (упрощённый режим восстановления).

### Неделя `/week`

Сводка текущей недели (понедельник — воскресенье):

- **Недельный босс** — условия, прогресс, награда;
- **XP за неделю** — факт / цель и статус (Слабая → Сильная неделя);
- **Календарь дней** — XP по каждому дню, отметки калорий/шагов/алкоголя;
- **Недельные бонусы** — доп. XP за серии (зал, трезвость, калории, замеры в понедельник).

### Отчёты `/reports`

**Еженедельные отчёты** с итогами:

- статус недели (от «Тяжёлой» до «Идеальной»);
- XP и монеты за неделю;
- дни в лимите калорий, с выполненными шагами, без алкоголя;
- зал, шаги, лучший день;
- изменение веса/талии;
- лучшая привычка, краткое резюме и совет.

На **главной** в понедельник появляется превью отчёта за прошлую неделю.

### Замеры `/measurements`

Ввод и история **замеров тела**:

- вес, грудь, талия, живот, бёдра, бедро, бицепс;
- график веса и талии;
- дельта от первого замера;
- **бонус XP** за замер в понедельник (настраивается).

Замеры влияют на аватар, карту прогресса, достижения и отчёты.

### Навыки `/skills`

Шесть навыков, каждый качается от своих действий:

| Навык | Иконка | За что растёт |
|-------|--------|--------------|
| **Тело** | 💪 | Шаги, зал, зарядка |
| **Контроль** | 🎯 | Калории, замеры |
| **Ясность** | 🧠 | Трезвость, дневник |
| **Быт** | 🏠 | Готовка, ремонт |
| **Зелень** | 🌱 | Уход за растениями |
| **Радость** | 🎨 | Хобби |

У каждого навыка — уровень, полоска XP и описание.

### Карта прогресса `/map`

Пять **путей** с вехами (майлстоунами):

| Путь | Что отслеживает |
|------|-----------------|
| ⚖️ Путь веса | Сброшенные кг от стартовой точки |
| 💧 Путь ясности | Самая длинная серия трезвых дней |
| 👟 Путь шагов | Суммарные шаги |
| 🏋️ Путь тренировок | Визиты в зал |
| 📏 Путь замеров | Регулярные замеры |

На карте видно пройденные и ближайшие вехи.

### Аналитика `/insights`

**Локальная умная аналитика** — подсказки на основе ваших данных (без облака):

- позитивные наблюдения («шаги растут 2 недели подряд»);
- замечания («мало дней без алкоголя»);
- советы по улучшению.

Фильтры: все / позитивные / замечания / советы.  
Для работы нужно накопить достаточно записей за несколько дней.

### Награды `/rewards`

**Магазин мотивации** за монеты:

| Вкладка | Описание |
|---------|----------|
| **Доступные** | Награды, которые можно купить |
| **Купленные** | Уже полученные награды |
| **Монеты** | История начислений и трат |
| **Банк** | «Копилка» — отложенные монеты |
| **Добавить** | Создать свою награду (название, описание, цена) |

Примеры наград: «Кино», «Десерт», «Новая вещь» — всё настраивается под себя.

### Достижения `/achievements`

Коллекция **ачивок** с уровнями редкости (бронза → легендарная):

- категории: вес, замеры, калории, шаги, алкоголь, тренировки, дневник, комбо, боссы, восстановление, XP;
- фильтр по категории и статусу (все / полученные / не полученные);
- при разблокировке — **всплывающий тост** на любом экране.

### Настройки `/settings`

Все параметры приложения. После изменений нажмите **«Сохранить»** (кроме темы).

| Блок | Что настраивается |
|------|-------------------|
| **Внешний вид** | Тема: «Светлая уютная» или «Тёмное фэнтези» (сохраняется сразу) |
| **Персонаж прогресса** | Целевой вес (кг) |
| **Аватар** | Пол, авто/ручной режим стадии, пороги кг для этапов 1–7 |
| **Цели по умолчанию** | Калории, шаги, зал/неделя, XP/неделя |
| **Цели по неделям** | Индивидуальные цели с даты начала недели |
| **Монеты 🪙** | Сколько монет за хороший день, неделю, бонусы |
| **Баллы (XP)** | Сколько XP за каждый квест и недельный бонус |
| **Второстепенные цели** | Скрыть, переименовать, добавить свои; иконка и цвет карточки |
| **Бэкап** | Скачать файл базы `.sqlite` |

## Игровые механики

### Квесты дня

**Основные** — три столпа дня:

- калории ≤ лимита → XP;
- шаги ≥ цели → XP;
- без алкоголя → XP; умеренный/тяжёлый алкоголь → штраф XP.

**Второстепенные** — чекбоксы-привычки с XP и (для встроенных) прокачкой навыков.

Лимиты калорий/шагов/зала берутся из **цели недели** (ближайшая `weekStart` ≤ текущей даты) или из значений по умолчанию.

### XP и уровни

XP начисляется за:

- выполненные квесты дня;
- недельные бонусы (зал, трезвость, калории, замеры);
- победу над боссом.

**Уровни** (базовые пороги): 1 → 2 (500 XP) → 3 (1200) → 4 (2200) → 5 (3500)…

Все значения XP настраиваются в **Настройки → Баллы (XP)**.

### Монеты

Монеты — отдельная валюта для **магазина наград**:

- хороший / отличный день;
- героический день, «железный» день;
- недельные пороги 80% / 100%;
- бонусы за трезвую неделю, зал, калории, идеальную базу;
- замер в понедельник, минимальный день восстановления.

Настраиваются в **Настройки → Монеты**.

### Навыки

Каждое действие даёт **skill XP** в соответствующий навык. При заполнении полоски — повышение уровня навыка.  
Кастомные второстепенные цели дают XP, но **не прокачивают навыки** (только встроенные привычки связаны с навыками).

### Недельный босс

Каждую неделю — случайный **босс** с набором условий, например:

- N дней без алкоголя;
- M тренировок;
- X% недельного XP;
- дни в лимите калорий.

**Победа** → бонус XP и монеты. **Идеальное выполнение** → усиленная награда.

### Восстановление после плохого дня

Если вчера был **плохой день** (мало XP, тяжёлый алкоголь, сильный перебор калорий):

- на «Сегодня» показывается карточка **восстановления**;
- основные квесты заменяются на **минимальный набор** (упрощённые цели);
- за минимальный день можно получить монеты — без давления.

### Достижения

Разблокируются автоматически при выполнении условий (серии дней, сброс веса, комбо-привычки, победы над боссами и т.д.).  
Прогресс проверяется при загрузке и сохранении данных.

### Карта прогресса

Пять независимых **путей** с вехами. Каждая веха — конкретное число (кг, дни, шаги, тренировки).  
Показывает, где вы сейчас и какая цель следующая.

### Отчёты и аналитика

- **Отчёты** — структурированный итог каждой недели;
- **Аналитика** — живые инсайты и тренды по вашим данным.

Обе системы работают **локально**, без отправки данных наружу.

### Аватар и персонаж

- **7 стадий** персонажа (мужской / женский);
- в **авто-режиме** стадия = максимальный этап, где сброс веса ≥ порога;
- в **ручном** — выбираете стадию сами;
- картинки: `public/avatars/{male,female}/` (см. [docs/AVATAR_ASSETS.md](docs/AVATAR_ASSETS.md)).

### Темы оформления

| Тема | Стиль |
|------|-------|
| **Светлая уютная** 🌤️ | Светлая, мягкая, уютная |
| **Тёмное фэнтези** 🌙 | Тёмная RPG с фиолетовыми и золотыми акцентами |

Переключение: **Настройки → Внешний вид**. Сохраняется в localStorage и в базе.

### Свои второстепенные цели

**Настройки → Второстепенные цели** (под блоком «Баллы»):

1. **Скрыть** встроенную цель (например, «Хобби»).
2. **Изменить** встроенную — название, описание, иконка, цвет карточки, XP.
3. **Добавить свою** — например, «Йога» 🧘 с бирюзовым фоном и 5 XP.
4. **Удалить** свою цель.

Встроенные по умолчанию:

| Цель | Категория |
|------|-----------|
| ☀️ Зарядка | Средняя |
| ✍️ Дневник | Средняя |
| 🏋️ Зал | Средняя |
| 🍳 Готовка | Бонусная |
| 🔧 Ремонт | Бонусная |
| 🌱 Цветы | Бонусная |
| ✨ Хобби | Бонусная |

## Установка и запуск

### Разработка

**Требования:** Node.js 18+, PHP 8+, npm.

```powershell
cd E:\Работа\OSPanel\domains\personal-rpg
npm install
npm run build

# Терминал 1 — API
.\scripts\dev-api.ps1

# Терминал 2 — фронт с hot reload
npm run dev
```

Откройте **http://localhost:5173** (прокси `/api` → `:8080`).

| Команда | Описание |
|---------|----------|
| `npm run dev` | Dev-сервер Vite |
| `npm run build` | Проверка TypeScript + сборка в `dist/` |
| `npm run preview` | Просмотр production-сборки |

### Продакшен на Windows

1. PHP в `server/runtime/php/`
2. nginx в `server/runtime/nginx/` — см. [server/AUTOSTART.md](server/AUTOSTART.md)
3. `npm run build`
4. Проверка: `.\scripts\start-all.ps1` → **http://127.0.0.1:8080**
5. Автозапуск: двойной клик **`install-autostart.bat`** (или PowerShell от администратора: `.\scripts\install-autostart.ps1`)

**Без PowerShell вручную:** `PersonalRPG.bat` (старт + браузер) / `PersonalRPG-Stop.bat` (стоп) / `PersonalRPG-Autostart.bat` (только сервер, для автозапуска)

**Порты:** 8080 (сайт), 9000 (PHP-CGI).

### Установщик для другого ПК

1. На машине разработчика: `.\scripts\build-release.ps1`
2. Собрать `installer/personal-rpg.iss` в [Inno Setup](https://jrsoftware.org/isinfo.php)
3. Получить `release/PersonalRPG-Setup.exe`

**Node.js на целевом ПК не нужен.** Подробнее: [installer/README.md](installer/README.md).

## Структура проекта

```
personal-rpg/
├── api/                 # PHP API + SQLite
│   ├── index.php        # REST-эндпоинты
│   ├── Database.php     # Схема и миграции
│   └── bootstrap.php    # Хелперы, маппинг строк БД
├── data/                # SQLite (не в git)
├── dist/                # Собранный фронт (после npm run build)
├── public/avatars/      # Картинки аватара
├── scripts/             # PowerShell: dev, автозапуск, релиз
├── server/              # nginx, PHP runtime, автозапуск
├── src/
│   ├── pages/           # Экраны приложения
│   ├── components/      # UI-компоненты
│   ├── constants/       # Достижения, навыки, темы, привычки
│   ├── utils/           # Движки: квесты, XP, боссы, инсайты…
│   ├── store/           # Zustand-стор
│   └── types/           # TypeScript-типы
└── README.md
```

---

## Технологии

| Слой | Стек |
|------|------|
| **Frontend** | React 19, TypeScript, Vite 6, Tailwind CSS 4, Zustand, React Router, Recharts, Lucide |
| **Backend** | PHP 8, SQLite |
| **Сервер** | nginx + PHP-CGI (Windows) |
