Place per-chapter art here. Filenames must match `src/constants/journeyChapterVisuals.ts`:

```
chapter-01-ruins.webp
chapter-02-crack.webp
chapter-03-road.webp
chapter-04-tower.webp
chapter-05-ridge.webp
chapter-06-lake.webp
chapter-07-fortress-order.webp
```

Recommended size: **640×360** or **480×270** (16:9 vignette).

If a file is missing, the biome CSS gradient still renders — art is optional.
