# Journey chapter backgrounds

Place chapter vignette backgrounds here.

- chapter-01-ruins.webp

- chapter-05-pass.webp
- chapter-06-lake.webp

Recommended format:

- `.webp` (`.png` / `.jpg` supported as fallback)
- landscape orientation
- 16:9 or close
- no embedded text
- no numbers
- no UI frame
- no logo/watermark

All captions, chapter numbers and statuses are rendered by the React UI.

Config: `src/constants/journeyChapterVisuals.ts`
